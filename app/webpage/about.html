STRUTTURA

Analisi

L'obiettivo principale del progetto condotto da "Sushi Squad" è la creazione di uno strumento che permetta la conversione fra i vari codici appresi durante il corso di "Tecnologie e Progettazione di Sistemi Informatici di Telecomunicazioni".

Progettazione

Per massimizzare la compatibilità tra i vari codici abbiamo scelto di usare una base comune (base 10), ove possibile, e di effettuare il passaggio dei valori attraverso stringhe.

Codici implementati:

  • Codici alfanumerici
    • Romano
    • Base N
    • ASCII
    • UTF-8
  • Codici pesati
    • BCD
    • Aiken
    • Quinario
    • Biquinario
    • 2 su 5
  • Codici non pesati
    • Eccesso 3
    • Gray
    • Eccesso 3 Riflesso
    • 1 su N
  • Altro
    • 7 segmenti
    • Codice fiscale

Codifica

Il programma è stato diviso in due parti principali:

  • La parte backend contenente gli algoritmi effettivi programmati in C++
  • La parte frontend riguardante l’interfaccia grafica scritta in HTML e CSS

Infine, come “colla”, per permettere la comunicazione tra le due parti è stato utilizzato JavaScript. Ciò è stato possibile grazie al framework Electron, che ha permesso di creare moduli operativi da codice sorgente in C++, richiamabili da JavaScript. Per controllare gli input ed effettuare il parsing delle stringhe sono state usate espressioni regolari (RegEx). Infine, per l’implementazione del codice a 7 segmenti è stato usata la libreria di p5.js che grazie a semplici comandi JavaScript ha permesso la creazione del canvas.

GLOSSARIO

  • JavaScript, abbreviato js, è un linguaggio di programmazione orientato agli oggetti e agli eventi, comunemente utilizzato nella programmazione Web lato client (esteso poi anche al lato server) per la creazione, in siti web e applicazioni web, di effetti dinamici interattivi tramite funzioni di script invocate da eventi innescati a loro volta in vari modi dall'utente sulla pagina web in uso (mouse, tastiera, caricamento della pagina ecc...).
  • Node.js è una runtime di JavaScript Open source multipiattaforma orientato agli eventi per l'esecuzione di codice JavaScript, costruita sul motore JavaScript V8 di Google Chrome. Molti dei suoi moduli base sono scritti in JavaScript, e gli sviluppatori possono scrivere nuovi moduli in JavaScript.
  • Electron (precedentemente noto come Atom Shell) è un framework open source gestito e ospitato da GitHub. Consente lo sviluppo della GUI di applicazioni desktop utilizzando tecnologie Web: combina il motore di rendering Chromium e il runtime Node.js.
  • Espressione regolare, o regex, è una sequenza di simboli (quindi una stringa) che identifica un insieme di stringhe.
  • p5.js è una libreria JavaScript, per artisti, disegnatori e insegnanti, concentrata soprattutto sulle arti visive. È un modo estremamente facile per creare opere d'arte interattive, animazioni e prototipi nel browser.
